# FizzBuzz

This is a simple Python script that prints the numbers from `1` to `n`, replacing certain numbers based on divisibility rules.

# Usage

Run the script and input a number `n` to display the FizzBuzz sequence from 1 to `n`.

## FizzBuzz Rules
```python
- If the number is divisible by 3 → print "Fizz"
- If the number is divisible by 5 → print "Buzz"
- If the number is divisible by both 3 and 5 → print "FizzBuzz"
- Otherwise, print the number itself
```

## Example
### input
```
Enter a number: 15
```
