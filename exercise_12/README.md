Create a class named `Circle` that represents the geometric shape of a circle. The class should be able to calculate 
the area and the perimeter (circumference) of a circle based on its radius.

Requirements:

Attributes:

    radius: A decimal number representing the radius of the circle.
    
Methods:

    area(): Should return the area of the circle, calculated with the formula:
            area = π × radius²

    perimeter(): Should return the perimeter (circumference) of the circle, calculated with the formula:
            perimeter = 2 × π × radius

Use the value of π provided by the math module (`math.pi`)
