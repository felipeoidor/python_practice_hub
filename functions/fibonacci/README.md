# Generate the Fibonacci Sequence Up to `n` Terms

This is a simple Python script that prints the first `n` Fibonacci numbers using memoization to improve performance by avoiding redundant calculations.

## Usage

Run the script and enter how many Fibonacci numbers you want to see: 

## The Fibonacci sequence is defined as:

```
- Fibonacci(0) = 0
- Fibonacci(1) = 1
- Fibonacci(2) = 1
- Fibonacci(n) = fibonacci(n-1) + fibonacci(n-2) for n ≥ 2
```

## Example
### Input
``` 
Enter how many Fibonacci numbers to display: 6
```
### Output
```
Fibonacci (0) = 0
Fibonacci (1) = 1
Fibonacci (2) = 1
Fibonacci (3) = 2
Fibonacci (4) = 3
Fibonacci (5) = 5
```
